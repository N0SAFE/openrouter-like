## Popular References

### Common Error Codes

| Code | Description | Common Solution |
|------|-------------|-----------------|
| 401  | Unauthorized | Check your API key |
| 403  | Forbidden | Verify your account has access to the requested model |
| 404  | Not Found | Check the endpoint URL and model name |
| 429  | Too Many Requests | Implement rate limiting or upgrade your plan |
| 500  | Server Error | Check status page or contact support |
